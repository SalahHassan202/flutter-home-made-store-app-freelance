# 🛍️ Home Made Store App

A modern Flutter store landing page application built with clean UI components and responsive design principles to deliver a smooth and elegant shopping experience 📱✨

The application focuses on building a visually appealing store interface with reusable widgets, organized structure, and scalable Flutter UI development 🚀

This project was developed as a freelance-style Flutter UI application to practice building responsive and modern shopping layouts using reusable components and organized project architecture 🎨

The app focuses on:

# 🚀 Getting Started

Clone the repository 📦

```bash
git clone https://github.com/SalahHassan202/flutter-home-made-store-app-freelance.git
```

Go to project folder 📂

```bash
cd flutter-home-made-store-app-freelance
```

Install dependencies ⚙️

```bash
flutter pub get
```

Run the app ▶️

```bash
flutter run
```
